Sequence during PC assembly ...

Standard components

First you have to assemble the standard components

To protect the components you should ground yourself!

  1. Assemble processor
  2. Mount cooler
  3. Install memory
  4. Install power supply
  5. Mount mainboard
  6. Connect power, data and front cables

Multimedia cards

Then install the following cards:

  • Install graphics card
  • Install sound card

Drives

Then the drives follow:

  1. Install hard disk
  2. Install optical drives

Connect cables

The components must be connected to each other.

  1. Lay cables
  2. Connect case fan
  3. Connect drives with mainboard
  4. Connect DVD drive with power cable
  5. Connect hard drives with power cable
